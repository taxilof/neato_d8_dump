# Neato D8 RAM Dump Extraction Summary

## Source
- 12x 16MB RAM dumps from 0x40000000 to 0x4B000000+
- Total: ~188 MB concatenated into neato_ram.bin

## Key Findings

### Boot Loader (U-Boot 2020.04)
- **Version**: U-Boot 2020.04-1.7.0-2749_08020951_cfae4f98+gffc3fbe7e5
- **Console**: ttymxc1, 115200 baud
- **Silent mode**: enabled (silent=1)
- **DDR**: 2 GB (0x40000000-0xC0000000)

### Boot Flow (from U-Boot env)
1. `bootcmd` -> `reliable_boot`
2. MMC device 2, partition 1 or 2 (dual-boot A/B)
3. Load `key_blob.enc` from eMMC to `keyblob_addr=0x45000000`
4. Load+decrypt files from `enc_file_addr=0x46000000`:
   - `Image.enc` -> kernel to `loadaddr=0x40480000`
   - `*.dtb.enc` -> DTB to `fdt_addr=0x43000000`
   - `initrd.uImage.enc` -> initrd to `initrd_addr=0x43800000`
   - `swu-pubk.pem.enc` -> SWU public key to `swuk_addr=0x43100000`
   - Signature file
5. Boot with `booti ${loadaddr} ${initrd_addr} ${fdt_addr}`

### Encryption
- **CAAM** (Cryptographic Acceleration and Assurance Module) used for decryption
- Key blob loaded to 0x45000000, encrypted files at 0x46000000
- Decryption via U-Boot `file_decrypt` command
- In initramfs: `caam-keygen import /mnt/fs_key_blob.enc randomkey`
- Key stored at: `/data/caam/randomkey` (in kernel keyring as `logon:logkey:`)
- Rootfs decrypted with dm-crypt: `dmsetup create encrypted ... crypt capi:tk(cbc(aes))-plain :36:logon:logkey:`

### Kernel
- **Version**: Linux 5.4.47-rt28-1.7.0-2933 (RT preempt kernel)
- **Compiler**: GCC 9.2.0 (Yocto/OpenEmbedded)
- **Platform**: i.MX8M Nano (imx8mn)
- **Image**: ARM64, little endian, 4k page size
- **Size**: ~24 MB compressed
- **Config**: IKCFG_ST format, kernel config extracted

### Certificates (CSF/IMG secure boot)
- **CSF1_1_cert.der**: CN=CSF1_1_sha256_2048_65537_v3_usr, issued by SRK1_sha256_2048_65537_v3_ca
- **IMG1_1_cert.der**: CN=IMG1_1_sha256_2048_65537_v3_usr, issued by SRK1_sha256_2048_65537_v3_ca
- **silex_cert.der**: CN=silex (self-signed)
- All use RSA 2048-bit, SHA-256
- Valid 2020-08-17 to 2120 (100 years)

### Initramfs
- **Size**: ~40 MB compressed gzip
- **Format**: CPIO SVR4 no CRC
- **OS**: "Lego Startup Initrd" v0.1
- **Login**: login.shadow, sulogin, su (shadow-based auth)
- **Key commands**: caam-keygen, keyctl, dmsetup, fw_printenv

### Full Disk Encryption (FDE)
- Root partition encrypted with dm-crypt
- AES-CBC mode via CAAM (capi:tk(cbc(aes)))
- Key derived from key_blob.enc stored on boot partition
- Key blob format: "OgaT" magic + hex data (for old 4.14 kernels)
- Newer format imports directly via caam-keygen

### File Paths in eMMC (from boot env)
- **key_blob.enc**: / on mmc 2:1 (boot partition)
- **Image.enc**, **initrd.uImage.enc**, **fdt.enc**: / on mmc 2:1
- **Root**: /dev/mmcblk2p3 (boot1) or /dev/mmcblk2p4 (boot2)

### Device Trees (5 DTBs found)
- **dtb_0x1FCDC0**: 1039 bytes (small, likely OTA/TF-A related)
- **dtb_0x2BDBB0**: 28851 bytes (Neato main DTB)
- **dtb_0x3000000**: 36301 bytes (main SoC DTB, largest)
- **dtb_0x3200000**: 802 bytes (small overlay or sub-node)
- **dtb_0x8000000**: 1039 bytes (copy of first)

### Splash Screen
- **bitmap.bmp**: 160x96, 8-bit color, Windows 3.x format, 15538 bytes
